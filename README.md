# FlutterStudy

Flutter学习练手项目

## 功能技术点

* APP基础框架搭建
* 主要页面保活
* pageView、自定义CustomScrollView、瀑布流布局
* 路由、命名路由以及参数传递、转场动画
* 下拉刷新、上拉加载更多
* 本地图片、网络图片、读取json文件、json转模型
* webView使用，安卓上面webView https和http混合模式使用
* 国际化，根据系统语言切换APP语言
* 暗黑模式，切换主题
* 状态共享
